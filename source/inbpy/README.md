# Notebook Bundle (`source/inbpy`)

Notebook-focused bundle for Kaggle / interactive runs.

- `kaggle_multigpu_modeling.ipynb`: multi-GPU notebook (T4 x2 friendly).
- `kaggle_multigpu_modeling_optimized.ipynb`: optimized standalone notebook (cross-table features, COGS-ratio branch, horizon-bucket blend, SHAP).

You can upload this notebook directly to Kaggle and run end-to-end.
